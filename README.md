# My First Free Website

This is a beginner website made with only HTML, CSS, and JavaScript.

## What each file means

- `index.html` is the main website page.
- `assets/css/style.css` controls colors, layout, spacing, and design.
- `assets/js/main.js` controls the small "back to top" button.
- `assets/images/profile-card.png` is the main image used on the homepage.

## How to open on this laptop

Double-click `index.html`.

## Free website address

You do not need to buy a domain right now.

Use GitHub Pages for a free address like:

```text
https://yourgithubusername.github.io
```

Later, when you have money for a domain, you can connect a name like:

```text
yourname.com
```

## What to change first

Open `index.html` and search for:

```text
Your Name
yourname@example.com
@yourusername
```

Replace those with your real details.

## Upload using GitHub without command line

1. Go to https://github.com
2. Create a new repository named `yourgithubusername.github.io`
3. Upload everything inside this folder.
4. Go to `Settings` then `Pages`.
5. Choose `Deploy from a branch`.
6. Choose branch `main` and folder `/root`.
7. Save.

After a few minutes your site will be live at:

```text
https://yourgithubusername.github.io
```
